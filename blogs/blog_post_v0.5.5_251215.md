# Sync Multi Chat v0.5.5 업데이트: 대화 히스토리 저장 & 관리

**작성일**: 2025-12-15
**버전**: v0.5.5

**Sync Multi Chat v0.5.5** 릴리즈를 발표하게 되어 기쁩니다! 이번 업데이트는 많은 분들이 기다리셨던 **대화 히스토리 저장** 기능이 드디어 추가되었습니다. 이제 소중한 대화 내용을 잃어버릴 걱정 없이, 언제든지 이전 대화를 다시 불러와 작업을 이어갈 수 있습니다.

## 🆕 새로운 기능

### 1. 대화 히스토리 저장 (Conversation History)
이제 모든 대화 세션이 로컬 데이터베이스(IndexedDB)에 안전하게 저장됩니다. 왼쪽 사이드바를 통해 이전 대화 목록을 확인하고, 클릭 한 번으로 해당 시점의 작업 환경을 완벽하게 복원할 수 있습니다.

-   **자동 저장**: 대화가 진행될 때마다 자동으로 히스토리가 업데이트됩니다.
-   **완벽한 상태 복원**: 히스토리를 클릭하면 당시 활성화되었던 **AI 서비스**, **레이아웃**, **각 서비스의 URL** 상태가 그대로 복원됩니다.
-   **사이드바 인터페이스**: 왼쪽 상단의 햄버거 메뉴(☰)를 클릭하여 히스토리 사이드바를 열고 닫을 수 있습니다.

### 2. 히스토리 관리
저장된 대화 기록을 손쉽게 관리할 수 있는 기능들도 함께 제공됩니다.

-   **개별 삭제**: 더 이상 필요 없는 특정 대화 기록을 개별적으로 삭제할 수 있습니다.
-   **전체 삭제**: "Clear All" 버튼을 통해 오래된 기록을 한 번에 정리할 수 있습니다.
-   **안전한 삭제**: 삭제 시 확인 팝업을 띄워 실수로 중요한 기록을 지우지 않도록 보호합니다.

### 3. 반응형 사이드바
작업 공간을 효율적으로 활용할 수 있도록 사이드바 디자인을 개선했습니다.

-   **접이식 구조**: 평소에는 숨겨두었다가 필요할 때만 펼쳐서 넓은 화면을 유지할 수 있습니다.
-   **직관적인 UI**: 최신 대화 순으로 정렬되어 원하는 기록을 빠르게 찾을 수 있습니다.

## 🛠️ 사용자 경험 개선

### 성능 최적화
-   대화 기록이 많아져도 앱 성능에 영향을 주지 않도록 **비동기 데이터베이스 처리**를 구현했습니다.
-   앱 시작 시 마지막 세션을 불러오는 속도가 더욱 빨라졌습니다.

## 📋 다음 계획
저희는 사용자 피드백을 바탕으로 계속해서 앱을 발전시키고 있습니다:
-   **히스토리 검색**: 저장된 대화 내용을 키워드로 검색하는 기능
-   **히스토리 내보내기/가져오기**: 백업 및 기기 간 이동을 위한 기능
-   **태그/폴더 정리**: 대화를 주제별로 분류하는 기능
