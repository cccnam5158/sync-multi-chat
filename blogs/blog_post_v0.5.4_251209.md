# Sync Multi Chat v0.5.4 업데이트: 세션 저장 & URL 바 추가

**작성일**: 2025-12-09
**버전**: v0.5.4

**Sync Multi Chat v0.5.4** 릴리즈를 발표하게 되어 기쁩니다! 이번 업데이트는 **세션 유지**, **WebView URL 바**, **외부 링크 처리** 등 사용자 워크플로우를 크게 개선하는 기능들을 추가했습니다.

## 🆕 새로운 기능

### 1. 이어서 작업하기 - 세션 유지
이제 앱을 닫았다가 다시 열어도 **정확히 마지막 상태에서** 계속할 수 있습니다:
-   **사용 서비스 기억**: ChatGPT, Claude, Gemini 중 어떤 것을 사용 중인지 저장
-   **레이아웃 유지**: 1x3, 3x1, 1x4, 2x2 레이아웃 선택이 유지됩니다
-   **현재 URL 복원**: 각 AI 서비스에서 마지막으로 본 페이지로 바로 이동
-   **설정 보존**: 익명 모드, 스크롤 동기화 등 설정이 그대로 유지

💡 **팁**: Alt+F4로 강제 종료해도 상태가 저장됩니다!

### 2. URL 바 - 현재 페이지 정보를 한눈에
각 AI 패널 상단에 새로운 URL 바가 추가되었습니다.

| 버튼 | 기능 |
|------|------|
| **새로고침** | 해당 패널만 새로고침 (회전 애니메이션 피드백) |
| **URL 복사** | 현재 페이지 URL을 클립보드에 복사 |
| **대화 복사** | 전체 대화 스레드를 마크다운으로 복사 |
| **외부 브라우저** | 현재 페이지를 기본 브라우저에서 열기 |

긴 URL은 자동으로 말줄임 처리되며, 마우스를 올리면 전체 URL을 확인할 수 있습니다.

### 3. 외부 링크 자동 처리
AI가 답변에서 외부 웹사이트 링크를 제공하면 어떻게 하시나요? 이제 걱정 없습니다:
-   YouTube, Wikipedia, 블로그 등 **외부 링크 클릭 시 기본 브라우저에서 열기**
-   AI 서비스 내부 페이지는 앱 안에서 정상 작동
-   작업 흐름을 끊기지 않고 참고 자료를 편하게 확인

### 4. 3x1 세로 레이아웃
3개의 AI 서비스를 **세로로 나란히** 배치하는 새로운 레이아웃이 추가되었습니다.
-   화면이 가로로 넓은 환경에서 유용
-   각 서비스의 전체 너비를 활용 가능
-   기존 1x3(가로)과 3x1(세로) 중 선택

## 🛠️ 사용자 경험 개선

### 버튼 피드백 강화
-   **복사 버튼**: 클릭 시 녹색 체크마크(✓)가 2초간 표시되어 복사 완료 확인
-   **새로고침 버튼**: 클릭 시 아이콘이 빙글빙글 회전하여 동작 확인

### 더 부드러운 패널 조절
-   패널 크기 조절 시 **헤더, URL 바, 콘텐츠가 함께** 매끄럽게 이동
-   구분선이 평상시 얇게(1px), 마우스 호버 시 두껍게(3px) 표시되어 쉽게 잡기 가능

### 익명 모드 개선
-   익명 모드 활성화 시 헤더의 서비스 이름이 "ChatGPT (A)", "Claude (B)" 형태로 변경
-   앱 재시작 시에도 익명 모드 상태 유지
