# Sync Multi Chat v0.9.0: Left Panel 강화 + Prompt Hub 연결 + Mermaid 라이브 프리뷰 개선

**작성일**: 2026-03-27  
**버전**: v0.9.0

**Sync Multi Chat v0.9.0**은 프롬프트 작업 동선을 더 빠르게 만들기 위한 릴리스입니다.  
이번 버전은 **Left Panel 내비게이션 확장**, **커스텀 프롬프트 카테고리 관리 강화**, **Mermaid 라이브 프리뷰 안정화**에 집중했습니다.

---

## Left Panel에 Dashboard / Prompt Hub 추가

### 무엇이 달라졌나요?

- 좌측 패널에 **Dashboard** 메뉴를 추가해 시작 지점 접근성을 높였습니다.
- 좌측 패널에 **Prompt Hub** 메뉴를 추가해 프롬프트 관리 화면으로 바로 이동할 수 있습니다.
- 기존 **Chat History** 메뉴는 시인성을 개선해 더 빠르게 찾을 수 있도록 조정했습니다.

### 왜 중요한가요?

기존에는 기능 진입 경로가 분산되어 있어 반복 작업 시 클릭 비용이 컸습니다.  
이제 자주 쓰는 화면을 좌측 패널에서 즉시 전환할 수 있어 프롬프트 실험 루프가 짧아집니다.

---

## 커스텀 프롬프트 카테고리 관리 기능 강화

### 변경 사항

- 커스텀 프롬프트 빌더에서 카테고리 생성/정리 흐름을 강화했습니다.
- 루트/서브 카테고리 구조 기반으로 프롬프트를 더 체계적으로 분류할 수 있습니다.
- Prompt Hub와의 카테고리 연계 경험을 개선해 탐색과 정리가 자연스럽게 이어집니다.

### 기대 효과

프롬프트가 많아질수록 제목 검색만으로는 한계가 있었는데,  
이제 카테고리 중심 관리로 재사용성과 유지보수성이 좋아집니다.

---

## Mermaid 라이브 프리뷰 개선

### 변경 사항

- Mermaid 블록 프리뷰의 렌더링 안정성을 개선했습니다.
- 전체화면/맞춤(Fit) 전환에서 레이아웃 깨짐을 줄이고 재렌더 동작을 보강했습니다.
- 복잡한 다이어그램에서도 미리보기 신뢰도를 높였습니다.
