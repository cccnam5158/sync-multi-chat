# Sync Multi Chat v0.9.1: Prompt Hub·커스텀 프롬프트 카테고리 편집 버그 픽스

**작성일**: 2026-03-30  
**버전**: v0.9.1

**Sync Multi Chat v0.9.1**은 **Prompt Hub**와 **커스텀 프롬프트 빌더(Custom Prompt Builder)**에서 카테고리를 다룰 때 겪을 수 있는 **편집·드래그 앤 드롭** 관련 문제를 집중 수정한 패치 릴리스입니다.

---

## 카테고리 트리: 단일 클릭 vs 더블클릭(이름 변경)

### 무엇이 달라졌나요?

- 카테고리를 **한 번 클릭**해 필터할 때와 **두 번 클릭**해 이름을 바꿀 때가 서로 **경합**하지 않도록 했습니다.
- 단일 클릭으로 목록 필터를 적용하기 전에 **짧은 지연**을 두어, 더블클릭 시 **인라인 편집**이 열리기 전에 화면이 먼저 갱신되는 현상을 막았습니다.
- **Prompt Hub**와 **CPB 사이드 패널** 모두 동일한 원칙으로 맞췄습니다.

### 왜 중요한가요?

이름만 살짝 바꾸려다 목록이 먼저 바뀌면 입력 포커스와 트리 상태가 어긋나 **작업이 끊기는** 느낌이 났습니다.  
이제 **이름 변경 의도**와 **필터 의도**가 UI 레벨에서 분리됩니다.

---

## 커스텀 프롬프트 빌더: 그리드 행 → 카테고리 드래그 복구

### 변경 사항

- **Grid.js**가 내부적으로 다시 그릴 때(정렬 등) **행의 드래그 속성**이 빠지던 문제를 **재동기화**와 **MutationObserver**로 보완했습니다.
- Chromium에서 `<tr draggable>`이 기대대로 동작하지 않는 경우를 대비해 **셀(`td`) 단위**로 드래그를 허용하는 방식을 유지·보강했습니다.
- 모달에 **위임된 `dragstart`**를 두어, 목록 영역에서도 **프롬프트 ID·다중 선택**이 카테고리 드롭에 안정적으로 전달됩니다.

### 기대 효과

카테고리로 프롬프트를 끌어다 놓는 흐름이 **정렬·재렌더 이후에도** 끊기지 않습니다.

---

## 카테고리 드래그 앤 드롭 규칙 정리

### 변경 사항

- 루트/서브 카테고리의 **순서 변경**, **다른 부모로 이동**, **루트로 승격** 등 **드롭 규칙**을 일관되게 정리했습니다.
- **자기 자신**이나 **미분류(Uncategorized)** 위에 떨어뜨릴 때는 **의미 없는 변경이 일어나지 않도록** no-op 처리했습니다.
- 프롬프트를 카테고리 위로 끌 때 **드롭 대상 하이라이트**로 어디에 놓이는지 알아보기 쉽게 했습니다.

## 버전별 변경 요약

| 구분 | 내용 |
|------|------|
| **버그 수정** | 카테고리 단일/더블 클릭 경합 제거 (Hub + CPB) |
| **버그 수정** | CPB Grid.js 재렌더 후 프롬프트→카테고리 DnD 유지 |
| **버그 수정** | 카테고리 DnD 재정렬·재부모·no-op 경계 정리 |
| **개선** | 드롭 하이라이트, 트리 갱신 후 인라인 이름 입력 포커스 |
