Sync Multi Chat v0.8.2 업데이트 안내

이번 버전은 Gemini 세션 안정성을 대폭 개선했습니다.

1. 대화 컨텍스트 유지
유휴 새로고침 시 새 챗으로 이동하지 않고 현재 대화 URL을 그대로 리로드합니다. 멀티 패널에서 진행 중이던 대화가 사라지지 않습니다.

2. 프롬프트 입력 중 자동 일시정지
하단 메인 입력에서 텍스트를 쓰는 동안에는 Gemini 새로고침 타이머가 자동으로 일시정지(깜빡임)됩니다. 3초간 입력이 없으면 다시 시작합니다.

3. 스크롤·커서 위치 복원
새로고침 후에도 대화 스크롤 위치와 메인 입력의 커서 위치가 유지됩니다.
