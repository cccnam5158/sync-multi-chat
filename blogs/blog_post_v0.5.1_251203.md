# Multi-AI Chat v0.5.1 업데이트: 익명 교차 검증 및 모던 UI

**날짜**: 2025-12-03
**버전**: v0.5.1

**Multi-AI Chat v0.5.1** 릴리스를 발표하게 되어 기쁩니다! 이번 업데이트는 새로운 프라이버시 중심 모드로 **교차 검증(Cross Check)** 기능을 강화하고, 사용자 인터페이스를 더욱 현대적이고 간결하게 다듬는 데 중점을 두었습니다.

## 🚀 새로운 기능

### 1. 익명 교차 검증 (Anonymous Cross Check)
Multi-AI Chat의 가장 강력한 기능 중 하나는 한 AI의 답변을 다른 AI에게 입력하여 검토하게 하는 "교차 검증"입니다. 하지만 답변의 출처(예: "이것은 ChatGPT가 말한 것입니다...")를 알면 모델의 반응에 편향이 생길 수 있습니다.

v0.5.1에서는 **익명 교차 검증**을 도입했습니다:
-   **기능**: "Anonymous" 토글을 활성화하면 시스템이 프롬프트에서 AI 서비스 이름을 중립적인 별칭(예: **(A)**, **(B)**, **(C)**)으로 자동 대체합니다.
-   **중요성**: 이를 통해 AI가 답변을 생성한 모델의 *평판*이 아닌 답변의 *내용* 자체를 비판적으로 평가하도록 보장합니다.
-   **사용 방법**: 교차 검증을 실행하기 전에 Cross Check 버튼 옆의 "Anonymous" 스위치를 켜기만 하면 됩니다.

### 2. 모던 UI 개편 (Modern UI Overhaul)
인터페이스를 더 깔끔하고 반응성 있게 다듬었습니다:
-   **스크롤 동기화 토글**: 스크롤 동기화 기능이 이제 일반 버튼 대신 세련된 토글 스위치를 사용합니다.
-   **아이콘 기반 레이아웃 컨트롤**: 텍스트 기반 레이아웃 버튼(1x3, 1x4, 2x2)을 직관적인 SVG 아이콘으로 교체하여 공간을 절약하고 미적 감각을 높였습니다.
-   **컴팩트한 디자인**: 버튼 높이와 글꼴 크기를 최적화하여 채팅 패널을 위한 화면 공간을 더 확보했습니다.
-   **색상 구분**: "새 대화(New Chat)" 및 "대화 내용 복사(Copy Chat Thread)"와 같은 주요 작업에 고유한 색상을 적용하여 한눈에 쉽게 알아볼 수 있게 했습니다.

## 🔧 개선 사항
-   **시각적 일관성**: 새로운 UI 요소들이 현대적인 디자인 표준(shadcn/ui 미학에서 영감)과 더 잘 어우러집니다.
-   **사용성**: 새로운 토글과 아이콘으로 제어 바의 시각적 복잡성을 줄였습니다.

## 🔜 다음 계획
저희는 계속해서 다음 기능들을 작업하고 있습니다:
-   **응답 비교 뷰(Diff View)**: AI 응답 간의 차이점을 강조하는 시각적 도구.
-   **DeepSeek 및 Copilot 지원**: 지원되는 AI 서비스 목록 확장.
