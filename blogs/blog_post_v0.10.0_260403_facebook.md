Sync Multi Chat v0.10.0 업데이트 안내

이번 버전은 macOS 정식 지원, 앱 내 로그인 흐름 강화, ChatGPT 구독 기반 Task·Skills가 핵심입니다.

1. macOS
Apple Silicon(M1/M2/M3…)용 arm64 DMG와 Intel용 x64 DMG를 각각 제공합니다. GitHub Releases에서 CPU에 맞는 파일을 받으세요.

2. 앱 내 로그인
Google/OAuth 단계를 외부 Chrome에만 의존하지 않고, 앱 안 모달(웹뷰와 동일 세션)에서 처리하는 경로가 강화되었습니다.

3. ChatGPT 구독 + Skills
Plus/Pro/Team 구독으로 OAuth 연결 후 Task UI에서 활용하고, 번들된 Skills(src/data/skills)를 도구형 워크플로에 사용할 수 있습니다.

Windows 사용자는 자동 업데이트 또는 아래 릴리즈의 x64 설치 파일을 이용하세요.
mac 사용자는 arm64/x64 DMG 중 선택해 설치하세요.
