# Sync Multi Chat v0.5.7 ~ v0.5.13 업데이트: 자동 업데이트, 히스토리 일괄 삭제, 로그인 안정화

**작성일**: 2026-01-08
**버전**: v0.5.7 ~ v0.5.13

**Sync Multi Chat**의 대규모 업데이트 소식을 전해드립니다! v0.5.7부터 v0.5.13까지 7개 버전에 걸쳐 **자동 업데이트 시스템**, **히스토리 일괄 삭제**, **로그인 안정화**, **UX 개선** 등 다양한 기능이 추가되고 개선되었습니다.

---

## 🚀 주요 하이라이트

### 1. Windows 인스톨러 & 자동 업데이트 (v0.5.8)

더 이상 수동으로 다운로드할 필요가 없습니다!

-   **Windows 인스톨러(.exe)**: 기존 포터블 버전 대신 정식 인스톨러로 배포됩니다. 설치 마법사를 통해 간편하게 설치하세요.
-   **자동 업데이트**: 앱 시작 시 새 버전을 자동으로 확인하고, 클릭 한 번으로 업데이트할 수 있습니다.
-   **GitHub Actions CI/CD**: 릴리스 자동화로 더 빠른 배포가 가능해졌습니다.

### 2. 업데이트 UX 개선 (v0.5.12 ~ v0.5.13)

-   **다운로드 진행률 창**: 업데이트 다운로드 시 진행률 바와 함께 속도, 파일 크기가 표시되는 전용 창이 뜹니다.
-   **작업 표시줄 진행률**: Windows 작업 표시줄에도 다운로드 진행률이 표시됩니다.
-   **릴리스 노트 깔끔하게**: 업데이트 알림에서 HTML 태그 없이 깔끔한 텍스트로 변경 사항을 확인할 수 있습니다.

---

## 📚 히스토리 관리 강화 (v0.5.10)

### Selection Mode로 일괄 삭제

히스토리 사이드바에 **Selection Mode**가 추가되었습니다.

-   체크박스로 여러 세션을 선택
-   **All / Clear** 버튼으로 전체 선택/해제
-   **Delete** 버튼으로 선택한 세션들을 한 번에 삭제 (삭제 전 확인 다이얼로그)

### UX 개선

-   **무한 스크롤 안정화**: 대용량 히스토리에서도 부드러운 스크롤
-   **스크롤 위치 복원**: 목록을 다시 열어도 이전 스크롤 위치가 유지됩니다
-   **모던 UI**: 선택 도구 모음의 스타일이 pill 버튼 형태로 개선되었습니다

---

## 🔐 로그인 안정화 (v0.5.7 ~ v0.5.13)

### Google 기반 서비스 로그인 수정

Gemini, Genspark 등 Google 로그인 기반 서비스의 세션 유지 문제가 해결되었습니다.

-   **accounts.google.com 쿠키 동기화** (v0.5.11): Gemini 재시작 시 로그인이 풀리던 문제 수정
-   **쿠키 이름 감지 개선**: `_Secure-*` 및 `__Secure-*` 형식 모두 인식
-   **외부 Chrome 로그인 수정** (v0.5.13): 앱 내에서 로그아웃 후 Chrome으로 재로그인해도 세션이 유지됩니다
-   **Chrome 프로필 쿠키 처리**: 만료된 쿠키로 인한 잘못된 로그인 감지 방지

### Grok 로그인 개선 (v0.5.7)

-   **Google/X 쿠키 동기화**: Grok 로그인 상태가 더 안정적으로 유지됩니다

### 쿠키 동기화 강화 (v0.5.9)

-   **SameSite 매핑**: Electron과 Chrome 간의 쿠키 정책 차이 해결
-   **기본 경로 설정**: ChatGPT/Claude/Gemini/Grok 세션이 Windows 11에서도 안정적으로 유지됩니다

---

## 💾 Custom Prompt 영구 저장 (v0.5.9)

재설치해도 프롬프트가 사라지지 않습니다!

-   **electron-store 저장**: 사용자 정의 프롬프트가 앱 데이터 폴더에 영구 저장됩니다
-   **자동 마이그레이션**: 기존 localStorage 데이터가 자동으로 이전됩니다
-   **크롬 로그인 배지 수정**: 외부 로그인 창을 닫으면 "Chrome으로 로그인" 버튼이 정상적으로 리셋됩니다

---

## 🎨 UI/UX 개선 (v0.5.11)

### 버튼 재배치

-   **새로 고침 버튼**: URL 바에서 헤더(확대 버튼 왼쪽)로 이동하여 접근성 향상
-   **서비스별 New Chat 버튼**: 각 서비스 패널 URL 바에 플러스(+) 아이콘 버튼 추가
    -   개별 서비스만 새 채팅으로 리셋 가능
    -   다른 서비스에 영향 없음
