# Brand Hub — Sync Multi Chat

## Brand Summary

Sync Multi Chat is an open-source, non-commercial desktop app that broadcasts one prompt to multiple official AI chat web UIs (including ChatGPT, Claude, Gemini, Grok, and Perplexity) so users can compare responses side-by-side without API keys. It provides multi-pane layouts, scroll sync, anonymous comparisons, thread/markdown copying, and AI cross-checking to speed up multi-model evaluation workflows.

## Detailed Description

Sync Multi Chat is an open-source, non-commercial desktop application that helps users compare outputs from multiple AI chat services side-by-side by broadcasting the same prompt to each service at the same time. It targets the workflow problem of having to open and manage multiple browser tabs and manually copy/paste prompts when evaluating answers across models like ChatGPT, Claude, Gemini, Grok, and Perplexity. The core value proposition is faster, more consistent multi-model comparison using the official web interfaces without requiring API keys.

The product is built around a flexible multi-pane UI that can run up to four AI services concurrently in layouts such as 2x2, 1x3, 1x4, and vertical arrangements. A central input bar enables "Simultaneous Prompting" (type once, send everywhere). The app adds comparison-oriented tooling including Scroll Sync (one panel scrolls all panels), Anonymous Mode (hide AI names during comparisons to reduce bias), and "Markdown Copy" to copy entire threads or only the last response in one click. It also includes "AI Cross-Check," which triggers each service to analyze and compare other services' responses.

Sync Multi Chat positions itself as "privacy first" by using direct login to official services, avoiding a middleman architecture, and not storing credentials. It states that sessions run in isolated sandboxed environments and that "Session Persistence" automatically restores layouts, active services, and specific URLs. The app supports multi-model workflows with "Universal File Support," enabling drag-and-drop or paste of images and text, plus uploading to multiple AIs with a verified two-step workflow.

Distribution is via downloads for supported platforms, with Windows currently offered (v0.7.1 installer with auto-update) and macOS and a Chrome extension listed as "coming soon." The website emphasizes that the project is open-source and non-commercial, suggesting adoption is driven by power users who want orchestration and cross-checking across the "most capable AI models" rather than an API-based enterprise integration approach.

## Persona — Ideal Customer

**AI Power User / Model Evaluator**

The primary users are AI power users who regularly work across multiple AI chat services and need to compare outputs quickly and consistently. They are typically knowledge workers, developers, researchers, analysts, writers, or prompt engineers who evaluate different models for accuracy, style, and completeness and want to reduce the overhead of managing multiple tabs and manual copy/paste. Likely decision-makers are individual users who self-serve by downloading the app (especially on Windows), and who value workflow features like multi-pane layouts, scroll synchronization, copying responses as markdown, and keeping sessions persistent across restarts. They are also privacy-sensitive users who prefer direct login to official services with no credential storage and no API-key-based integration. Use is global/online, oriented around services such as ChatGPT, Claude, Gemini, Grok, and Perplexity, and includes handling text and image inputs that may need to be uploaded to multiple AIs at once.
