# Brand Hub — Sync Multi Chat

## 브랜드 요약

Sync Multi Chat은 하나의 프롬프트를 여러 공식 AI 채팅 웹 UI(ChatGPT, Claude, Gemini, Grok, Perplexity 등)에 동시에 전송해, API 키 없이 응답을 나란히 비교할 수 있게 해 주는 오픈소스·비상업용 데스크톱 앱이다. 멀티 패널 레이아웃, 스크롤 동기화, 익명 비교, 스레드/마크다운 복사, AI 크로스체크를 제공해 멀티 모델 평가 워크플로를 가속한다.

## 상세 설명

Sync Multi Chat은 동일한 프롬프트를 여러 AI 채팅 서비스에 동시에 보내 한 화면에서 응답을 나란히 비교할 수 있게 하는 오픈소스·비상업용 데스크톱 애플리케이션이다. ChatGPT, Claude, Gemini, Grok, Perplexity 등 여러 모델의 답변을 평가할 때 브라우저 탭을 여러 개 열고 프롬프트를 수동으로 복사·붙여넣기하는 워크플로 문제를 해결하는 데 초점을 둔다. 핵심 가치는 **API 키 없이 공식 웹 인터페이스를 사용해 더 빠르고 일관된 멀티 모델 비교**를 제공하는 것이다.

제품은 2×2, 1×3, 1×4, 세로 배치 등 레이아웃으로 최대 4개 AI 서비스를 동시에 실행할 수 있는 유연한 멀티 패널 UI를 중심으로 구성된다. 중앙 입력창에서 “한 번 입력, 모든 서비스로 전송”이 가능하다. 비교에 특화된 기능으로 **스크롤 동기화**(한 패널 스크롤 시 모든 패널 동기화), **익명 모드**(비교 시 AI 이름 숨김으로 편향 감소), **마크다운 복사**(전체 스레드 또는 마지막 응답만 한 번에 복사), **AI 크로스체크**(각 서비스가 다른 서비스의 응답을 분석·비교)를 제공한다.

Sync Multi Chat은 공식 서비스에 직접 로그인하고 중간 서버 없이 동작하며 자격 증명을 저장하지 않는 **프라이버시 우선** 구조를 내세운다. 세션은 격리된 샌드박스 환경에서 실행되며, **세션 지속성**으로 레이아웃, 활성 서비스, 특정 URL을 자동 복원한다. **범용 파일 지원**으로 이미지·텍스트 드래그 앤 드롭/붙여넣기와 여러 AI에 업로드하는 검증된 2단계 워크플로를 지원한다.

배포는 지원 플랫폼용 다운로드로 이루어지며, 현재 Windows용(v0.7.1 인스톨러, 자동 업데이트 지원)이 제공되고, macOS와 Chrome 확장은 “coming soon”으로 안내된다. 프로젝트는 오픈소스·비상업용이며, API 기반 엔터프라이즈 통합보다는 “가장 강력한 AI 모델” 간 오케스트레이션과 크로스체킹을 원하는 파워 유저 중심 채택을 전제로 한다.

## 주요 기능 (3가지 이상)

| 구분 | 내용 |
|------|------|
| **핵심** | Multi-AI Prompt Broadcaster — 한 프롬프트를 여러 AI 서비스에 동시 전송 |
| **UI** | 유연한 멀티 패널 UI로 최대 4개 서비스 동시 실행 |
| **레이아웃** | 2×2, 1×3, 1×4, 세로/특화 레이아웃 지원 |
| **비교** | 패널 간 스크롤 동기화로 나란히 읽기 |
| **비교** | 익명 모드로 비교 시 AI 서비스 이름 숨김 |
| **복사** | 마크다운 복사(전체 스레드 또는 마지막 응답만) |
| **분석** | AI 크로스체크 — 각 AI가 다른 AI 응답 분석·비교 |
| **세션** | 레이아웃·활성 서비스·URL 자동 복원(세션 지속성) |
| **보안** | 프라이버시 우선, 공식 서비스 직접 로그인(중간자 없음) |
| **접근** | API 키 불필요, 공식 웹 UI 사용 |
| **격리** | 격리된 샌드박스 환경에서 세션 실행 |
| **파일** | 범용 파일 지원(이미지·텍스트 드래그 앤 드롭/붙여넣기) |
| **업로드** | 검증된 2단계 프로세스로 다중 AI 업로드 |
| **배포** | Windows 인스톨러 + 자동 업데이트(v0.7.1) |
| **라이선스** | 오픈소스·비상업용 배포 모델 |

## 페르소나 — 이상적 고객

**AI Power User / Model Evaluator**

주요 사용자는 여러 AI 채팅 서비스를 자주 쓰고, 출력을 빠르고 일관되게 비교해야 하는 AI 파워 유저다. 지식 노동자, 개발자, 연구자, 분석가, 작가, 프롬프트 엔지니어 등이 정확도·스타일·완성도 측면에서 모델을 평가하며, 여러 탭 관리와 수동 복사/붙여넣기 부담을 줄이고자 한다.

의사결정은 주로 개인 사용자가 앱을 직접 다운로드(특히 Windows)하는 셀프 서비스 형태이며, 멀티 패널 레이아웃, 스크롤 동기화, 마크다운 복사, 재시작 후에도 유지되는 세션 같은 워크플로 기능을 중시한다. 자격 증명 저장이나 API 키 기반 연동 없이 공식 서비스에 직접 로그인하는 방식을 선호하는 프라이버시 민감 사용자다. 이용은 전 세계·온라인 기준이며, ChatGPT·Claude·Gemini·Grok·Perplexity 등 서비스와 텍스트·이미지 입력을 다루고, 필요 시 여러 AI에 동시 업로드하는 사용 패턴을 포함한다.
