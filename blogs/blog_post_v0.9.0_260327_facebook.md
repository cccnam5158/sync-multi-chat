Sync Multi Chat v0.9.0 업데이트 안내

이번 버전은 프롬프트 작업 동선을 더 빠르게 만들었습니다.

1. Left Panel 강화
Dashboard, Prompt Hub 메뉴가 추가되어 원하는 화면으로 바로 이동할 수 있습니다.

2. Chat History 시인성 개선
좌측 메뉴에서 Chat History를 더 쉽게 찾고 접근할 수 있도록 UI 가독성을 높였습니다.

3. 커스텀 프롬프트 카테고리 관리 강화
카테고리 기반으로 프롬프트를 더 체계적으로 정리하고 Prompt Hub와 연계해 활용할 수 있습니다.

4. Mermaid 라이브 프리뷰 개선
Mermaid 블록의 렌더링 안정성, 전체화면/맞춤 동작, 재렌더 신뢰도를 보강했습니다.

기존 사용자라면 앱 재시작 후 "지금 업데이트"를 눌러주세요.
