Sync Multi Chat v0.8.1 업데이트 안내

이번 버전의 핵심은 두 가지입니다.

1. Perplexity 로그인 탐지 개선
로그아웃 화면(한·영 로그인 문구, 인증 URL 등)을 명확히 구분하고, 로그인 버튼은 화면에 보일 때만 반영해 Login Required 배지가 실제 세션과 더 잘 맞도록 했습니다.
DOM이 잠깐 바뀔 때 배지가 깜빡이지 않게 짧은 유예도 넣었습니다.

2. Edge만 설치된 PC 지원 (외부 로그인)
Chrome이 없으면 자동으로 Microsoft Edge(Chromium)로 외부 로그인·쿠키 동기화를 진행합니다.
Windows·macOS 모두 지원합니다.
