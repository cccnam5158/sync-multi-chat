왜 이 버전이 더 나은가

기존 스캐폴드는 화면이 풍부했지만, 사용자가 “무엇부터 눌러야 하는지”가 약했습니다. v2는 Entry → Quick Run → Running → Result → Builder 흐름을 상단 journey bar로 명확히 보여줍니다.

현재 추천 액션 `Cross Check / Automation`을 눌러 Automation Hub Lite에 들어가 보세요. Builder는 바로 노출하지 않고, 먼저 프리셋 3종의 가치를 경험하도록 설계했습니다.
